Latest Project

cigarette smoke detection device

The purpose of this tool is to detect when the driver smokes in the truck, the tool will detect cigarette smoke and send evidence in the form of photos that will be sent to a telegram. For the microcontroller itself uses ESP32-Wrover Cam which is used as a controller and functions as an image taker in the tool. Followed by the MQ-2 Sensor to detect cigarette smoke, LDR Sensor to detect light intensity, relay to turn on / off the LED when the room is less light when cigarette smoke is detected. And LED as a light intensity enhancer when around the room is dark.

PCB Desain

This project is made to design a pcb for a lamp dimmer circuit. Which dimmer itself is a tool to control the brightness of the lamp itself. For the controller itself, it uses a chip from ESP-32 which is combined with a dimmer circuit consisting of diac and triac.
